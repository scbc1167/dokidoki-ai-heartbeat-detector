# dokidoki-ai-heartbeat-detector
An AI/IoT prototype using a XIAO ESP32 and mmWave radar to measure heart rate non-contact. It visualizes “dokidoki” moments via LED blinking and motor vibration synced to the pulse. Built on FreeRTOS with tasks, queues, and buffers for sensing, display, and communication. Time-series data is analyzed by AI to generate real-time emotional messages.
